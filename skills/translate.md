# Translate

When the user asks for translation, translate the input between Chinese and English. Auto-detect the source language.

## Instructions

- Output format:
  > **Original:** (source text)
  > **Translation:** (translated text)
  > **Notes:** (ambiguous terms, alternative translations, or cultural context — omit if none)
- Preserve the tone and register of the original
- For technical terms, keep the original in parentheses after the translation
