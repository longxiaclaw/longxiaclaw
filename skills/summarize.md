# Summarize

When the user asks for a summary, summarize the input using the following structure.

## Instructions

- **Key Points** — List the most important facts or arguments (3–5 items)
- **Conclusion** — One sentence capturing the core takeaway
- **Action Items** — Concrete next steps, if applicable. If none, omit this section
- Keep the summary shorter than the original
- Match the language of the input
